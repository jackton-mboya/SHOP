API Documentation

Welcome to the API documentation for our service. Here you will find details on how to interact with our API endpoints.

Base URL

The base URL for accessing our API is: http://127.0.0.1:8000/api/

Endpoints

1. List and Create Subscribers

Endpoint: /api/subscribers/

Method: GET, POST

Description: Retrieve a list of subscribers or create a new subscriber.

Request Body (POST):

{
    "email": "string"
}

Response:

HTTP 200 OK
[
    {
        "id": 1,
        "email": "string",
        "subscription_date": "YYYY-MM-DDTHH:MM:SSZ"
    }
]

2. Retrieve, Update, or Delete Subscriber

Endpoint: /api/subscribers/<id>/

Method: GET, PUT, DELETE

Description: Retrieve, update, or delete a specific subscriber by ID.

Request Body (PUT):

{
    "email": "string"
}

Response:

HTTP 200 OK
{
    "id": 1,
    "email": "string",
    "subscription_date": "YYYY-MM-DDTHH:MM:SSZ"
}

3. List and Create Contact Messages

Endpoint: /api/contact_messages/

Method: GET, POST

Description: Retrieve a list of contact messages or create a new contact message.

Request Body (POST):

{
    "first_name": "string",
    "last_name": "string",
    "email": "string",
    "subject": "string",
    "message": "string"
}

Response:

HTTP 200 OK
[
    {
        "id": 1,
        "first_name": "string",
        "last_name": "string",
        "email": "string",
        "subject": "string",
        "message": "string",
        "responded": true
    }
]

4. Retrieve, Update, or Delete Contact Message

Endpoint: /api/contact_messages/<id>/

Method: GET, PUT, DELETE

Description: Retrieve, update, or delete a specific contact message by ID.

Request Body (PUT):

{
    "first_name": "string",
    "last_name": "string",
    "email": "string",
    "subject": "string",
    "message": "string",
    "responded": true
}

Response:

HTTP 200 OK
{
    "id": 1,
    "first_name": "string",
    "last_name": "string",
    "email": "string",
    "subject": "string",
    "message": "string",
    "responded": true
}
